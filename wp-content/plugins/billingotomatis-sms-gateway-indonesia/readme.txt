Module ini memungkinkan Anda untuk mengintegrasikan wordpress Anda dengan layanan SMS gateway Billingotomatis.com

== Description ==

Billingotomatis.com merupakan penyedia layanan penyedia otomatisasi seperti pengecekan mutasi ke bank-bank poluler seperti: BCA, MANDIRI, BNI, BRI, BNI (dan terus berkembang), serta menyediakan layanan SMS Gateway.

Kenapa pakai SMS? Karena tidak semua orang akan memeriksa email nya, jadi SMS akan lebih bernilai bagi bisnis Anda daripada mengirimkan notifikasi via E-Mail

== Installation ==

Sebelum memakai plugin ini pastikan Anda telah memiliki ioncube terinstall pada module PHP Anda.

== Frequently Asked Questions ==

= Ada pertanyaan seputar modul? =

Silahkan kunjungi https://www.billingotomatis.com/ untuk info dan pertanyaan lebih lanjut
